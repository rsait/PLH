# Poet's Little Helper source code

The Basque corpora are reduced versions of the original ones due to intellectual property issues.

For Basque lemmatization it is necesary to install some ixa-pipe tools:

* [ixa-pipe-tok](https://github.com/ixa-ehu/ixa-pipe-tok)
* [ixa-pipe-pos](https://github.com/ixa-ehu/ixa-pipe-pos)
